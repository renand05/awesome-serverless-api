This API has been implemented using Dependency Injection so that Inversion of Control is applied.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run build

# watch 
$ npm run start:dev

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
